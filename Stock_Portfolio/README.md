# 📊 Stock Portfolio Tracker (Python)

## 📌 Description
This project is a simple Python-based stock portfolio tracker that allows users to calculate the total value of their stock investments.

---

## 🎯 Objective
- Understand data handling in Python
- Work with user input
- Perform calculations

---

## ⚙️ Features
- 📥 Input stock names and quantities
- 💰 Calculate total investment value
- 📊 Display portfolio summary
- 💾 Option to save data (TXT/CSV)

---

## 🛠️ Technologies Used
- Python
- File handling
- Basic arithmetic operations

---

## ▶️ How to Run

1. Open terminal  
2. Navigate to folder  

Run:
python stock.py

---

## 💡 Example
Enter stock name: AAPL
Enter quantity: 2
Total: 360

---

## 🧠 How It Works
- User enters stock details
- Program calculates total cost
- Displays summary
- Optionally saves results
