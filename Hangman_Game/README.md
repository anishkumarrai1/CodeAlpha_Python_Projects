# 🎮 Hangman Game (Python)

## 📌 Description
Hangman is a classic word guessing game implemented in Python.  
The player tries to guess a hidden word by suggesting letters within a limited number of attempts.

---

## 🎯 Objective
- Practice Python fundamentals
- Work with loops and conditionals
- Build interactive terminal-based games

---

## ⚙️ Features
- 🎲 Random word selection
- 🔤 Letter-by-letter guessing
- ❌ Limited attempts (lives system)
- 📊 Displays progress after each guess
- 🔁 Replay option

---

## 🛠️ Technologies Used
- Python (Core)
- Random module
- Loops & conditionals

---

## ▶️ How to Run

1. Open terminal / command prompt  
2. Navigate to project folder  

Run:
python hungman.py

---

## 💡 Example Gameplay
Word: _ _ _ _ _ Guess a letter: a Correct guess!
Word: a _ _ _ a

---

## 🧠 How It Works
- A random word is selected
- User inputs letters
- Correct guesses reveal letters
- Wrong guesses reduce attempts
- Game ends when word is guessed or attempts run out
